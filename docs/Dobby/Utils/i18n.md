# i18n

**File:** `src/utils/i18n.js`

Internationalization system for 3 languages.

## Supported Languages

| Code | Language | Flag |
|------|----------|------|
| `en` | English | 🇬🇧 |
| `hi` | हिंदी (Hindi) | 🇮🇳 |
| `gu` | ગુજરાતી (Gujarati) | 🇮🇳 |

## Exports

### `t(key)`

Translation lookup. Falls back to English, then returns key itself.

### `getLang()` / `setLang(lang)`

Get/set current language. Persists to `localStorage('dobby-lang')`. Dispatches `dobby-lang-change` event.

### `SUPPORTED_LANGS`

Array of `{ code, label, flag }` objects.

## Event System

Language change dispatches:
```js
window.dispatchEvent(new CustomEvent('dobby-lang-change', { detail: lang }))
```

Components listen and update their `lang` state accordingly.

## Coverage

All UI strings are translated:
- Header (undo, redo, share, upgrade)
- Sidebar (labels, presets, controls)
- Canvas (view names, export buttons, status)
- Chat (placeholder, remaining calls)
- Login (title, tagline, buttons)
- Landing (CTA, skip, demo, community)
- Upgrade (title, features, pricing)
