# i18n

**File:** `src/utils/i18n.js`

Internationalization data and functions.

## Supported Languages

| Code | Language | Coverage |
|------|----------|----------|
| `en` | English | Source |
| `hi` | हिंदी (Hindi) | Full |
| `gu` | ગુજરાતી (Gujarati) | Full |

## String Categories

~50 translation keys covering:
- Header (undo, redo, share, upgrade)
- Sidebar (sett builder, weave, thread size, repeats, presets, gallery)
- Canvas (view names, export buttons, status)
- Chat (placeholder, send, remaining calls)
- Login (title, tagline, Google button, demo mode)
- Landing (enter, skip, demo request, community links)
- Upgrade (title, CTA, dismiss)

## Storage

Language preference stored in `localStorage('dobby-lang')`. Default: `en`.
