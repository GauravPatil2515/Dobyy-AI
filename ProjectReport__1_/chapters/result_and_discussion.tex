\section{System Output}
The Dobby Studio interface provides real-time visualization of fabric designs. User inputs are processed through the AI pipeline and displayed on an interactive HTML5 canvas. The system renders design changes within milliseconds, providing immediate visual feedback. The interface supports zooming and panning for detailed inspection of textile patterns.

The 3D drape simulation uses Three.js to render cloth physics, displaying how a designed fabric appears as draped material. The visualization updates parametrically with vertex displacement animations simulating realistic cloth behavior.
